# Platform

## Purpose

The object to be characterised is placed on this rotating platform.

##  Description

- **Type:** 3D print
- **Material Suggested:** PLA
- **Approx. Size:** 80mm x 80mm x 40mm
  
## Assembly Instructions

1. The 8mm shaft of the platform is inserted in the NEMA17 coupler, which in turn is connected to the motor controlling the theta DOF.

## Notes

A lnoger shaft for this piece would be better in a v2, since attaching it to the coupler was a struggle.
